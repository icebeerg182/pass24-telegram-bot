# Changelog

## [0.0.3] — 2026-07-17

### Добавлено

- Расширенный словарь сокращений марок (~80 брендов): Peugeot, Citroen, Opel, Fiat, Alfa Romeo, Seat, Cupra, Suzuki, Infiniti, Genesis, китайские бренды (Voyah, Aito, Nio, XPeng, Belgee и др.), американские (Dodge, Cadillac, Hummer) и др.

## [0.0.2] — 2026-07-03

### Добавлено

- Интерактивные настройки в `deploy/install.sh`:
  - спрашивать тип ТС (легковой/грузовой) при заказе
  - спрашивать подтверждение перед созданием пропуска
  - кнопка выбора адреса по умолчанию (если несколько адресов)
- Переменные `BOT_ASK_VEHICLE_TYPE`, `BOT_CONFIRM_BEFORE_CREATE`, `BOT_ENABLE_ADDRESS_PICKER`
- Кнопки «Легковой» / «Грузовой» и подтверждение заказа в Telegram
- Кнопка «📍 Адрес» для смены адреса по умолчанию

## [0.0.1] — 2026-07-03

### Добавлено

- Telegram-бот для заказа автомобильных пропусков PASS24 (mobile API жителя)
- Гибкий парсер марки и госномера (любой порядок, пробелы, две строки)
- Словарь сокращений марок и подсказки при ошибке
- Управление доступом: `/allow`, `/deny`, временное открытие `/open 12|24|48`, `/close`
- Кнопки «Изменить» и «Удалить» под созданным пропуском
- Docker-образ и `docker-compose.yml`
- Интерактивный установщик `deploy/install.sh` с проверкой кредов
- `deploy/validate_env.py` — проверка `.env` (Telegram + PASS24)
